# Industrial RUL Predictive Maintenance

Predictive Maintenance and Remaining Useful Life (RUL) estimation project using the NASA CMAPSS turbofan engine degradation dataset.

This repository explores how Machine Learning and operational intelligence can be combined to support industrial maintenance decision-making.

The project includes:

- Exploratory Data Analysis (EDA)
- Remaining Useful Life (RUL) estimation
- Operational degradation analysis
- Sensor behavior analysis
- Operational guard rails
- Health state classification
- Predictive Maintenance modeling
- Explainability and industrial interpretation

---

# Project Goals

The main objective is to build an industrial-style predictive maintenance workflow capable of:

- estimating remaining equipment lifetime;
- identifying degradation patterns;
- detecting operational anomalies;
- supporting maintenance prioritization;
- simulating operational engineering rules.

This project is intentionally designed with a production-oriented mindset rather than only a modeling notebook.

---

# Dataset

Dataset used:

NASA CMAPSS Turbofan Engine Degradation Simulation Dataset

The dataset contains:
- multiple engines;
- operational cycles;
- sensor measurements;
- progressive degradation until failure.
